About me

I’m a data-focused developer with a strong foundation in Python, SQL, and building practical data-driven applications.

My work sits at the intersection of data analysis and engineering. I enjoy taking raw, unstructured data and turning it into something useful—whether that’s through cleaning and transformation, building simple pipelines, or creating visualizations that highlight meaningful insights.

I started with web development, which helped me understand how users interact with applications, but I’ve shifted my focus toward working with data at scale. Recently, I’ve been strengthening my skills in data processing, analysis, and backend workflows to prepare for data analyst and junior data engineering roles.

I’m particularly interested in roles where I can:

  • Work with real datasets and solve practical problems
  • Build and optimize data workflows
  • Contribute to systems that support data-driven decisions

Tech stack: Python (Pandas, NumPy), SQL, JavaScript, Tableau, Git
